# Add two integers
**Compute the sum of two integers a and b.**

Input

- Line 1: contains two integers a and b (0 <= a, b <= $10^{19}$)

Ouput

- Write the sum of a and b
  
**Example1**

- Input: `3 5`

- Output: `8`
  
**Example2**

- Input: `10000000000000000000 10000000000000000000`

- Output: `20000000000000000000`
